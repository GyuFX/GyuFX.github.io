## 문제

Hello World!를 출력하시오.



## 제출

```c++
#include <stdio.h>

int main(){
    printf("Hello World!");
    return 0;
}
```



## Stdio.h가 뭘까?

#include는 코드에 라이브러리, 헤더를 참조하는 역할

[stdio.h wiki](https://ko.wikipedia.org/wiki/Stdio.h)

stdio.h은 Standard Input/Output library (표준입출력 라이브러리)의 약어로써, C 언어의 표준 라이브러리 함수의 매크로 정의, 상수, 여러 형의 입출력 함수가 포함된 헤더 파일이다.
